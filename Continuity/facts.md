# Continuity Bible

This document is the canonical reference for minor details that must remain consistent throughout the novel. If a detail is established in the text, log it here.

## How to Use This File

**When drafting:** Before describing something for the first time, check if it's already established. After introducing new details, add them here.

**When revising:** Search this file before changing descriptions. If you change something, update both the draft AND this file.

**AI assistants:** Consult this file before describing characters, places, or objects. Never contradict established facts. Update this file when new canonical details are introduced.
