# Final Versions

This folder contains approved, finalized files ready for publication or distribution.

## Finalization Checklist

Before moving a file here, confirm:

- [ ] All edit notes (`{{...}}`) processed and removed
- [ ] All author notes (`[[...]]`) removed
- [ ] Continuity verified across all sections
- [ ] Line-level proofread complete
- [ ] Character names consistent throughout
- [ ] Timeline makes sense
- [ ] Style guide conventions followed
- [ ] Any facts/references verified

## Process

1. Complete all revisions in `Drafts/`
2. Run through the checklist above
3. Copy the final file here
4. Remove draft numbering if desired (e.g., `01-chapter-one.md` â†’ `chapter-one.md`)
5. Tag the commit: `git tag v1-final`
